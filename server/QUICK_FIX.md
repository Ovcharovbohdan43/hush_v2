# Быстрое решение проблемы линкера

## Проблема: LNK1318 - лимит PDB файлов

Это известная проблема Windows линкера. Я добавил настройку в `Cargo.toml` для отключения debug информации в dev режиме.

## Решение применено

В `Cargo.toml` добавлено:
```toml
[profile.dev]
debug = false
```

Это уменьшит количество PDB файлов и должно решить проблему.

## Попробуйте снова:

```powershell
cargo clean
cargo build
```

## Альтернативные решения:

### 1. Используйте release сборку:
```powershell
cargo build --release
```

### 2. Если проблема сохраняется:
- Закройте все IDE и отладчики
- Перезапустите терминал
- Попробуйте снова

### 3. Временное решение - отключить debug полностью:
Если нужно, можно временно использовать release сборку для разработки:
```powershell
cargo run --release
```

## Примечание

Отключение debug информации означает:
- ✅ Меньше PDB файлов = нет ошибки линкера
- ✅ Быстрее компиляция
- ⚠️ Нет возможности отладки в dev режиме (но можно использовать release с debug)

Для отладки можно использовать:
```powershell
cargo build --release
# или включить debug обратно в Cargo.toml когда нужно
```
