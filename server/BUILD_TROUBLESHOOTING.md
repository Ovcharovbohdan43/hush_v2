# Решение проблем сборки

## Ошибка LNK1318: Unexpected PDB error

Эта ошибка возникает когда линкер Windows достигает лимита открытых PDB файлов (12 файлов).

### Решения:

1. **Закройте другие программы**:
   - Закройте Visual Studio, VS Code с отладчиком, или другие IDE
   - Закройте отладчики (WinDbg, etc.)

2. **Очистите и пересоберите**:
   ```powershell
   cargo clean
   cargo build
   ```

3. **Перезапустите терминал**:
   - Закройте PowerShell/CMD
   - Откройте новый терминал
   - Попробуйте снова

4. **Используйте release сборку** (меньше PDB файлов):
   ```powershell
   cargo build --release
   ```

5. **Отключите отладочную информацию** (временно):
   Добавьте в `Cargo.toml`:
   ```toml
   [profile.dev]
   debug = false
   ```

6. **Перезагрузите компьютер** (если ничего не помогает)

### Предупреждения (можно игнорировать):

- `dead_code` warnings - это нормально для неиспользуемого пока кода
- Они не мешают компиляции

### После исправления:

Попробуйте снова:
```powershell
cargo clean
cargo build
```

Если проблема сохраняется, попробуйте:
```powershell
cargo build --release
```
